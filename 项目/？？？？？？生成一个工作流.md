# 生成一个工作流：24小时内Youtube最新发布的相关视频

1）每天晚上10点定时触发；

2）设置节点：包含搜索关键词（可多个关键词组合为‘或’关系！），还有过滤条件，比如24小时内发布的视频、视频语言、视频的时长超过10分钟、视频的观看次数超过100人等；

3）推荐一款Youtube情报数据分析工具，在整个Youtube中，按照设置节点中的搜索关键词和过滤条件，以观看次数做降序排列，取出前10个视频的信息；

4）获取每个视频的标题、链接URL、时长、观看次数，汇总成为摘要汇总；

上传到Notion

5）发送到Gmail邮箱；

# 生成一个工作流：监视Youtube频道的最新发布视频

1）每小时轮询触发；

2）设置节点：包含所关注的Youtube频道ID（可多个频道ID组合为‘或’关系！）；

3）对所有Youtube频道ID遍历循环，分别找出每个Youtube频道1小时内最新发布的所有视频的信息；

4）获取每个视频的标题、链接URL、时长、观看次数，汇总成为摘要汇总；

5）发送到Gmail邮箱；
